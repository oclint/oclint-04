Roadmap

M.O. is far from perfect, and we have a long road ahead to help developers achieve high quality of code.

We have some cool ideas in mind that developers can see these in the future release of M.O., for people who are interested, we have a list of them-

  • Accept flexible configurations
  • Other reporter formats, like HTML and XML
  • Better user experiences
  • More rules
  • Interpret XCode project file and inspect the whole project

Please notice they are not prioritized, yet, and developers (like you) can join us to have a better community. We appreciate for your help in advance.
